# ErasmusVida Madrid

Sito statico responsive per un progetto universitario di SEO e Comunicazione Digitale.

## Pagine incluse

- `index.html`
- `alloggi.html`
- `costo-della-vita.html`
- `universita.html`
- `trasporti.html`
- `vita-sociale.html`
- `checklist-erasmus.html`
- `contatti.html`

## Pubblicazione con GitHub Pages

1. Carica tutti i file di questa cartella in un repository GitHub.
2. Vai in Settings > Pages.
3. Seleziona il branch principale e la cartella root.
4. Salva e attendi la pubblicazione.

Il sito non richiede build, database o dipendenze esterne.

## Note SEO

- Ogni pagina ha `title` e `description` specifici.
- La navigazione interna e' presente in HTML su tutte le pagine.
- Le immagini sono locali, compresse e dotate di `alt` e descrizione visibile.
- I contenuti sono pensati intorno alle keyword principali del progetto: Erasmus Madrid, Erasmus a Madrid, costo della vita a Madrid, case per studenti a Madrid, migliori quartieri per studenti a Madrid, universita a Madrid Erasmus, trasporti Madrid studenti, vita notturna Madrid studenti e checklist Erasmus Spagna.
